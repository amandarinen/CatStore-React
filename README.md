# CatStore React

A React-based version of CatStore, built as part B of the course **Gränssnittsutveckling**. The application fetches cats from an external API and presents them with a modern, responsive interface.

## Built with

- React + Vite
- React Router DOM
- React Bootstrap
- Font Awesome
- The Cat API

## Pages

| Page | Description |
|------|-------------|
| `/` | Home page |
| `/cats` | Browse cats |
| `/owner` | About us and contact info |
| `/cart` | Cart and checkout |

## Getting started

```bash
npm install
npm run dev
```
